# Retail Chatter Research

This project is a read-only research tool that analyzes aggregated public Reddit discussions about publicly traded companies.

## Purpose
- Observe short-term discussion trends and themes
- Analyze sentiment and narrative shifts in retail investor chatter
- Produce daily HTML summary reports

## Data Sources
- Reddit (public posts and comments only)
- Subreddits: r/stocks, r/investing, r/wallstreetbets

## Scope
- Rolling 48-hour window
- User-provided list of stock tickers
- Strict rate limits and content caps
